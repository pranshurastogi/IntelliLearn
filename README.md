# IntelliLearn

IntelliLearn is an innovative learning platform that leverages blockchain technology—through Educhain and OpenCampus—to teach blockchain fundamentals via engaging and interactive games. Explore various mini-games that not only entertain but also educate you on blockchain, decentralized identity, and Web3 concepts.

## Features

- **Multiple Interactive Games**: 
  - **Blockchain Basics Stickman** - Learn the fundamentals of blockchain via an archery game.
  - **Web3 Miner Challenge** - Catch coins to discover fun facts about Web3 and Educhain.
  - **OpenCampus Code Quest** - Pilot your spaceship to unlock secrets of OCID Connect.
  - **DAO Dungeon Escape** - Navigate a dungeon, collect tokens, and unlock the exit door with advanced on-chain logic.
- **OCID Authentication**: Secure login via OCID Connect for a decentralized identity experience.
- **Dynamic NFT Rewards**: Earn dynamic NFTs based on your game performance, with attributes that update in real time.
- **Educational & Engaging Content**: Learn blockchain fundamentals and advanced concepts through immersive gameplay.

## Games Overview

### Blockchain Basics Stickman
**Objective:**  
Learn blockchain basics through an archery game where you shoot arrows to hit targets.

**Mechanics:**  
Use adjustable angle and power sliders to aim and shoot.

### Web3 Miner Challenge
**Objective:**  
Move your bucket to catch falling coins.

**Mechanics:**  
Each coin collected reveals a fun fact about Web3, Educhain, or blockchain.

### OpenCampus Code Quest
**Objective:**  
Pilot your spaceship, collect tokens, and unlock the secrets of OCID Connect.

**Mechanics:**  
Navigate using arrow keys and avoid obstacles while collecting tokens.

### DAO Dungeon Escape
**Objective:**  
Traverse a dungeon grid to collect tokens and unlock the exit door.

**Mechanics:**  
Grid-based navigation and puzzle-solving with on-chain integration for dynamic challenges.

## OCID Integration

IntelliLearn integrates with **OCID Connect** to offer a decentralized authentication experience. Players log in using their OCID (Open Campus ID), which enables personalized experiences and secure on-chain credentials.

For details on setting up OCID integration, refer to the [OCID Connect documentation](https://github.com/opencampus-xyz/ocid-connect-js).
